# Forecast Desk

Windows-приложение для прогнозов по TradingView и отправки сигналов в Telegram.

## Первая версия умеет

- открывать TradingView внутри окна через WebView2;
- фиксировать прогноз со скриншотом графика;
- сохранять актив, биржу, таймфрейм, направление, стартовую цену и комментарий;
- создавать проверку через заданное количество баров или реакцию у временной линии;
- считать `TP %` от свечи линии: high/low/close или автоматически по типу свечи;
- отправлять прогноз в Telegram;
- проверять прогноз по текущей цене KuCoin или Binance;
- проверять цель и стоп по свечам за весь период прогноза;
- делать контрольный скриншот после наступления времени проверки;
- автоматически продлевать проверку, если прогноз не исполнился;
- отправлять результат проверки в Telegram;
- хранить журнал и скриншоты локально.

## Важная логика

TradingView используется для графика, рисования и скриншотов. Автопроверка результата берёт цену и свечи отдельно через API биржи, чтобы не пытаться распознавать цену по картинке.

Если цель и стоп попали в одну и ту же свечу, программа ставит статус `Нужна проверка`: по OHLC-свече нельзя честно понять, что было первым внутри этой свечи.

В режиме `Реакция у линии` программа сначала находит свечу, в которую попало время вертикальной линии. От этой свечи берется базовая цена для `TP %`, а окно реакции считается после закрытия этой базовой свечи.

## Запуск из проекта

```powershell
dotnet run
```

## Где данные

Настройки, журнал и скриншоты сохраняются в:

```text
%APPDATA%\ForecastDesk
```

## Как пользоваться

1. Выбери актив и таймфрейм прямо в TradingView.
2. Проверь, что справа автоматически заполнились `Биржа`, `Актив` и `Таймфрейм`.
3. Нарисуй прогноз в TradingView.
4. Выбери тип проверки: `Через бары` или `Реакция у линии`.
5. Для временного сигнала нарисуй вертикальную линию на графике, укажи точное время линии и нужный `GMT`.
6. Укажи окно реакции в барах: сколько свечей после свечи линии ждать движение `UP` или `DOWN`.
7. Заполни направление и `Описание прогноза`.
8. В режиме линии укажи `TP %`, например `1`; стоп можно добавить при необходимости.
9. Нажми `Зафиксировать прогноз`.
10. Оставь программу запущенной для автопроверки.

Если поля справа не успели обновиться после изменения графика, нажми `Взять с графика`.
Кнопка `Настройки` сверху задает TP по умолчанию и базу расчета от свечи линии.
